[!] DESCRIPTION:
LURAX VIP adalah sistem manajemen joki terintegrasi yang menggabungkan 
kecepatan eksekusi C++ dengan antarmuka web yang interaktif. Dirancang 
khusus untuk menangani misi joki tier tinggi (Immortal/Legend).

[!] KEY FEATURES:
1.  SWORD TRAIL AUTH: Masuk ke sistem dengan simulasi tebasan pedang Asta.
2.  SQL LIVE MONITOR: Pemantauan orderan secara real-time dengan UUID.
3.  SFX NOTIFICATION: Suara peringatan "TING" setiap ada transaksi baru.
4.  SYNC CALCULATOR: Kalkulator harga otomatis dengan diskon Anti-Magic.
5.  KERNEL INJECTION: Simulasi mesin C++ untuk sinkronisasi data server.

[!] INSTALLATION (WEB):
1.  Pastikan 3 file (index.html, style.css, script.js) berada dalam satu folder.
2.  Buka index.html di browser (Chrome/Edge direkomendasikan).
3.  Lakukan "Slash Down" pada layar intro untuk masuk ke Dashboard.

[!] COMPILATION (C++):
1.  Gunakan compiler GCC atau MSVC.
2.  Compile dengan perintah: g++ main.cpp LuraxKernel.cpp -o LuraxCommander.exe
3.  Jalankan .exe untuk masuk ke mode Kernel Terminal.

[!] EMERGENCY ALERT:
Jika SLOTS menunjukkan angka < 03, segera lakukan "EXECUTE MISSION" 
sebelum sistem masuk ke mode COOLDOWN.
